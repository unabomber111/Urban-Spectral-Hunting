### What is Urban Spectral Hunting?
Urban Spectral Hunting is an interactive Python game where the player has to catch ghosts in different city zones before they spread.  
It includes:

- Random ghost spawn in independent zones  
- Progressive score for each ghost caught  
- Final boss appears at **200 points**  
- Color-coded log to distinguish spawn, catches, and boss  
- Graphical interface built with **Tkinter**  

### How it works
1. A GUI window opens with:  
   - Score bar  
   - Color-coded event log  
2. Each zone spawns ghosts randomly  
3. Catching a ghost gradually increases your score  
4. At **200 points**, the boss appears and stops ghost spawn until defeated  
5. After the boss, you can continue catching ghosts or close the window manually  

### Technologies and libraries
- Python 3  
- Tkinter (GUI)  
- Threading (parallel zone management)  
- JSON (score saving)  
- Random and Time (ghost spawn and delays)
